# my-webbrowser

### docker build

```
docker build -t ubuntu20.04 .
```

### docker run

```
docker run -it --rm ubuntu20.04 /bin/bash
```

### compile

```
gcc main.c -I /usr/local/include/freetype2 \-I /usr/include/libpng16 -lfreetype -lm
```
